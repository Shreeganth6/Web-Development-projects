Use color palletes from https://colorhunt.co/

choosing color palletes is important for a good web design

Analogous color pallete : 2 colors that are closer to each other in color wheel -> for Logo design
Complementary color pallete : 2 colors that are opposite to each other in color wheel -> for the Entire page color pallete
